# AI 对话助手

## 项目简介
这是一个基于 Python Flask 开发的 AI 对话助手网页应用。用户可以通过简洁的网页界面与 AI 进行对话交互。

## 功能特性
- 支持实时对话交互
- 简洁美观的用户界面
- 支持历史对话记录
- 响应式设计，适配移动端
